รายละเอียดแต่ละโฟลเดอร์และไฟล์
- dataset/: โฟลเดอร์เก็บ dataset สำหรับฝึกโมเดล แบ่งออกเป็น cropped_images/ สำหรับเก็บภาพ และ annotation/ สำหรับเก็บ labels ที่ระบุตำแหน่งลูกดอกในแต่ละภาพ

- configs/:โฟล์เดอร์เก็บไฟล์คอนฟิกสำหรับการตั้งค่า environment ของ Anaconda ที่ระบุ dependencies และแพ็กเกจที่จำเป็นในการรันโปรเจค 

- models/: โฟลเดอร์สำหรับเก็บไฟล์ output ที่ได้จากการตรวจจับหรือการนับคะแนนลูกดอก

- README.md: ไฟล์อธิบายโปรเจคและวิธีการติดตั้งและใช้งาน

- requirements.txt: รายการ dependencies ที่ต้องติดตั้งสำหรับการทำงานของโปรเจค

โฟลเดอร์หลักที่เก็บสคริปต์การทำงานของโปรเจค โดยแต่ละไฟล์มีหน้าที่เฉพาะ ดังนี้ : 

- dataloader.py: สคริปต์ที่จัดการข้อมูล dataset โดยกำหนดรูปแบบและขนาดของข้อมูลที่ต้องการใช้ในแต่ละ batch สำหรับการฝึกโมเดล สามารถเลือก dataset ที่ต้องการได้ในไฟล์นี้ 

- combine_labels.py: เป็นสคริปต์ที่ใช้รวมไฟล์ label ของข้อมูล labels จากหลายๆ ชุดข้อมูลเข้าด้วยกัน เพื่อนำไปใช้ในการเทรนโมเดลเดียวกัน

- new_crop.py: สคริปต์สำหรับการตัดภาพ (cropping) โดยปรับขนาดของภาพให้เป็น 800x800 พิกเซล เพื่อให้สอดคล้องกับขนาดที่ต้องการใช้ในการประมวลผล

- train.py: สคริปต์สำหรับฝึกโมเดล YOLO โดยใช้ dataset ใหม่ 

- predict.py: เป็นสคริปต์ที่ใช้สำหรับทำการพยากรณ์หรือทำนายตำแหน่งของลูกดอกบนกระดานปาเป้า โดยใช้โมเดลที่ได้รับการฝึกมาแล้ว เพื่อตรวจจับและแสดงตำแหน่งลูกดอกในภาพ

- detect_vdo.py: สคริปต์สำหรับตรวจจับตำแหน่งของลูกดอกจากไฟล์วิดีโอ โดยทำการประมวลผลภาพทีละเฟรมเพื่อตรวจจับลูกดอกและระบุพิกัดบนกระดานปาเป้า

- dtvdo_cscore.py: สคริปต์สำหรับนับคะแนนตามกติกาสากลที่ใช้ในเกมปาเป้าแบบ 301 คะแนน โดยใช้ข้อมูลตำแหน่งของลูกดอกที่ตรวจจับได้จาก YOLO

-score_team.py: สคริปต์สำหรับนับคะแนนแบบทีม โดยกำหนดการสลับระหว่างทีมและนับคะแนนแต่ละทีมแยกกัน เพื่อให้สอดคล้องกับการเล่นแบบทีม

- betawebcam.py: สคริปต์สำหรับตรวจจับลูกดอกแบบ real-time โดยใช้กล้องเว็บแคม (webcam) ซึ่งจะทำการตรวจจับภาพจากกล้องโดยตรงและประมวลผลทันที
